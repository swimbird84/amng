# AMNG

동영상 작품 및 배우를 로컬에서 관리하는 데스크탑 앱.

---

## 개발 스펙

| 항목 | 내용 |
|---|---|
| 플랫폼 | Windows (Electron) |
| UI | React + TypeScript |
| 스타일 | Tailwind CSS v4 |
| DB | SQLite (better-sqlite3) |
| 빌드 도구 | electron-vite |
| 패키지 매니저 | npm |

### 개발 명령어

```bash
npm run dev        # 개발 서버 실행
npm run build      # 빌드
npm run package    # Windows 배포 패키지 생성 (release/ 폴더)
```

### 데이터 저장 위치

- **DB**: `%APPDATA%/amng/data.db` (SQLite)
- **이미지**: `%APPDATA%/amng/images/{works|actors}/{id}.{ext}`

## 탭 구성

### 대시보드

통계 및 분포 현황을 한눈에 확인하는 메인 화면.

- **최근 등록 작품 / 배우** — 최근에 추가된 항목 카드 목록
- **발매일 분포** — 연도 → 월 → 작품 카드 순으로 드릴다운
- **데뷔일 분포** — 연도 → 월 → 배우 카드 순으로 드릴다운 (나이·비율점수·대표태그 표시)
- **나이대 분포** — 10년 단위(초·중·후반) 그룹별 배우 목록
- **평점 분포** — 별점 구간별 작품 수, 클릭 시 해당 작품 목록
- **평점 분포 (배우)** — 평균점수 구간별 배우 목록
- **컵 분포** — 컵 사이즈별 배우 목록

---

### 랭킹

배우 랭킹을 카테고리별로 TOP 10 카드로 표시. 각 섹션마다 순서 반전 및 전체 목록 모달 제공.

- **평점 랭킹** — avg_score 기준
- **출연 작품 수 랭킹**
- **피지컬 랭킹** — 키·바스트·허리·힙 비율을 가중치 설정으로 계산한 복합 점수
- **부위별 랭킹** — 바스트 / 힙 / 허리 / 키 각각
- **즐겨찾기 랭킹**

---

### 작품

작품 목록 관리 탭.

- **목록 표시** — 표지 이미지 + 품번 + 별점 + 발매일 + 레이블 배지 + 대표태그/배우 카드
- **검색** — 품번 키워드 / 태그(AND·OR) / 배우 / 레이블 복합 필터, 결과 건수 표시
- **정렬** — 품번 / 별점 / 발매일 / 등록일 / 타이틀 (오름·내림차순)
- **즐겨찾기 필터**
- **작품 등록·수정** — 표지 이미지, 품번, 발매일, 레이블, 재생 경로(로컬 파일·URL 복수), 출연 배우, 태그, 별점, 타이틀/코멘트
- **삭제** — 다중 선택 드래그 삭제 모드
- **툴팁** — 카드 호버 시 상세 정보 리치 툴팁

---

### 배우

배우 목록 관리 탭.

- **목록 표시** — 프로필 사진 + 이름 + 평점 + 생년월일(나이) + 신체 치수
- **검색** — 이름 키워드 / 태그(AND·OR) 복합 필터, 결과 건수 표시
- **정렬** — 이름 / 평균점수 / 생년월일 / 데뷔일 / 출연수 / 피지컬점수 / 등록일
- **즐겨찾기 필터**
- **배우 등록·수정**
  - 프로필 사진
  - 신체 정보 (키·바스트·허리·힙·컵)
  - 생년월일 / 데뷔일
  - 10개 항목 점수 슬라이더 (face·bust·hip·physical·skin·acting·sexy·charm·technique·proportions)
  - 레이더 차트로 점수 시각화
  - 태그, 코멘트
- **배우 상세 모달** — 출연 작품 목록 포함
- **툴팁** — 카드 호버 시 상세 정보 리치 툴팁

---

### 레이블

제작사(makers) → 레이블(studios) 2단계 계층 구조 관리 탭.

- **버킷 필터** — [작품] / [레이블] 토글로 기준 전환
  - 작품 기준: 제작사 총 작품 수 범위별 버킷
  - 레이블 기준: 보유 레이블 수별 버킷
  - 각 버킷 버튼에 제작사 수 / 레이블 수 / 작품 수 표시
- **제작사 버튼 그리드** — 클릭으로 펼치기·접기, 색상 배지
- **레이블 버튼 그리드** — 제작사 선택 시 소속 레이블 표시
- **레이블 작품 목록** — 레이블 선택 시 연도별 작품 미니 카드
- **검색** — 제작사 키워드 / 레이블 키워드 동시 검색, 결과 건수 및 작품 수 표시
- **제작사·레이블 관리** — 추가·수정·삭제, 제작사 배정, 색상 설정, 품번 코드 관리
- 검색어·버킷 모드·선택 상태 세션 간 유지

---

### 태그

작품 태그 / 배우 태그를 카테고리별로 관리하는 탭.

- **태그 목록** — 전체 사용 수 / 대표태그 수 / 등록일 표시
- **정렬** — 이름 / 사용 수 / 등록일
- **태그 클릭** — 해당 태그가 붙은 작품 또는 배우 카드 목록 표시
- **카테고리 관리** — 태그를 카테고리로 묶어 정렬 순서 관리
- **태그 추가·수정·삭제**

---

## 공통 UX

- 모든 모달 ESC 키로 닫기
- 작품/배우 카드 클릭 시 상세 뷰 모달 (스택 방식, 작품↔배우 간 이동 가능)
- 다크 테마 고정 (bg-gray-900 기반)
- 각 탭의 검색·정렬·필터 조건은 localStorage로 세션 간 유지
